# Environment Setup
```bash
mkdir e2e-ml-project && cd e2e-ml-project
code .
```

**VSCode Extensions**
- Python
  - Python debugger
  - Python environment manager
- Jupyter
- rainbow brackets
- csv edit
- excel viewer
- Marp for VS Code

**Check python version**
```bash
python --version
```

**Python Virtual Environment**
```bash
python -m venv .venv
source .venv/bin/activate
python --version
pip --version
```

**Install Jupyter Notebook and run it**
```bash
pip install jupyter
# jupyter notebook
```
